# 金融数据助手 - 微信小程序原型

## 1. Concept & Vision

打造一款专业、精致且具有独特个性的金融数据小程序。摒弃传统金融软件冷冰冰的界面，采用**深邃的暗色系 + 科技感渐变**，营造专业投资氛围的同时保持视觉舒适度。整体风格融合"赛博朋克"与"极简金融"美学——深邃的深蓝/紫色背景配合霓虹绿/红作为涨跌标识，数据可视化精致优雅，让用户感受到金融数据的科技感与力量感。

## 2. Design Language

### Aesthetic Direction
- **Theme**: 深色科技金融风 (Dark Tech Finance)
- **Reference**: Bloomberg Terminal meets Cyberpunk
- **Mood**: 专业、冷静、前沿、高端

### Color Palette
```css

--bg-secondary: #111827;       /* 次级背景 */
--bg-card: #1a2332;           /* 卡片背景 */
--bg-card-hover: #232f42;     /* 卡片悬浮 */
--accent-primary: #6366f1;    /* 主色调靛蓝 */
--accent-secondary: #8b5cf6;  /* 次级紫色 */

--text-secondary: #94a3b8;    /* 次级文字 */
--text-muted: #64748b;        /* 淡化文字 */
--success: #10b981;           /* 上涨绿 */
--danger: #ef4444;           /* 下跌红 */
```

### Typography
- **Display Font**: "Orbitron" (数字、标题) - 科技感强
- **Body Font**: "Noto Sans SC" (中文内容) - 清晰易读
- **Mono Font**: "JetBrains Mono" (代码、数值) - 数据展示

### Spatial System
- 基础单位: 4px
- 页面边距: 16px
- 卡片间距: 12px
- 卡片内边距: 16px
- 圆角: 12px (卡片), 8px (按钮), 20px (大圆角)

### Motion Philosophy
- **Page Transitions**: 淡入 + 轻微上移, 300ms ease-out
- **Data Updates**: 数字跳动动画, 强调数据变化
- **Tab Switch**: 平滑滑动指示器, 200ms
- **Hover Effects**: 微妙的发光和阴影提升
- **Loading States**: 脉冲动画 + 渐变闪烁

## 4. Features & Interactions

### 功能1: 基金实时估值
- **自选基金列表**: 显示用户关注的基金，支持搜索
- **实时估值数据**: 模拟实时更新的估算净值和涨跌幅
- **估值排行榜**: 显示今日涨幅/跌幅最大的基金
- **交互**:
  - 点击基金卡片 → 展开详情（基金经理、规模、历史业绩）
  - 长按/右滑 → 删除自选
  - 下拉刷新数据

### 功能2: A股情绪分析 & ETF分析
- **情绪指数仪表盘**: 环形进度指示器显示恐慌/贪婪指数
- **市场情绪指标卡片**:
  - 涨停/跌停数量
  - 炸板率
  - 昨日涨停今日表现
  - 涨跌停比
- **ETF资金流向**:
  - 显示主要ETF的净流入/流出
  - 柱状图可视化
  - 分类查看（宽基、行业、跨境）

### 功能3: 龙虎榜
- **龙虎榜列表**:
  - 按日期筛选
  - 显示上榜原因（涨幅偏离、跌幅偏离、换手率等）
  - 机构席位 vs 营业部席位买卖数据
- **营业部龙虎榜**:
  - 实力营业部排行
  - 近期操作股票列表

## 5. Component Inventory

### 5.1 StatusBar
- 高度: 44px
- 背景: 渐变透明
- 包含: 时间（居中）、信号/电量图标（右侧）

### 5.2 TabBar
- 高度: 60px + safe-area
- 背景: 毛玻璃效果 `backdrop-filter: blur(20px)`
- 3个Tab: 图标 + 文字
- Active状态: 主色调高亮 + 底部指示器

### 5.3 FundCard
- 背景: `--bg-card`
- 圆角: 12px
- 内容: 基金代码、名称、估算净值、涨跌幅、趋势线
- States:
  - Default: 静态展示
  - Hover: 背景变亮 + 阴影提升
  - Loading: 骨架屏脉冲动画
  - Rising: 涨跌幅数字绿色
  - Falling: 涨跌幅数字红色

### 5.4 EmotionGauge
- 环形进度条: SVG实现
- 中心数字: 大号Orbitron字体
- 颜色渐变: 红色(0) → 黄色(50) → 绿色(100)
- 标签: 恐慌/中性/贪婪

### 5.5 MetricCard
- 用于情绪指标展示
- 图标 + 数值 + 标签
- 正负值颜色区分

### 5.6 ETFBanner
- ETF名称 + 代码
- 净流入/流出数值
- 小型柱状图显示占比

### 5.7 DragonTigerCard
- 股票代码、名称
- 上榜原因标签
- 机构/营业部席位数据对比
- 展开/收起详情

## 6. Technical Approach

### 框架选择
- **原型实现**: 纯HTML5 + CSS3 + Vanilla JS
- **设计**: 移动端优先，模拟微信小程序视觉和交互
- **数据**: 使用模拟数据（mock data）展示真实数据结构

### 文件结构
```
/
├── index.html          # 主页面
├── styles.css          # 全局样式
├── app.js              # 主逻辑
└── SPEC.md            # 规格说明文档
```

### 响应式策略
- 设计基于375px宽度（iPhone标准）
- 最大宽度480px居中显示
- 使用 `viewport-fit=cover` 适配刘海屏
