# AGENTS.md - 金融数据助手·微信小程序
## 项目定位
微信小程序原型，深色科技金融风格，包含三大核心模块：基金估值、A股情绪分析、龙虎榜。

## 核心约束（强制遵守）
1. **视觉风格**：严格使用文档指定的深色科技金融风、色值、字体、间距、动画。
2. **技术栈**：原生微信小程序（WXML + WXSS + JS + JSON），不使用框架。
3. **结构**：底部3个Tab固定，页面结构完全按原型实现。
4. **数据**：使用Mock模拟数据，数据结构严格遵循文档给出的JS结构。
5. **交互**：必须实现文档描述的所有动画、切换、展开、刷新、滑动效果。
